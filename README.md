# GPA & CGPA Calculator Application

A modern, responsive web-based GPA and CGPA calculator built with React, TypeScript, and Vite. This application helps students track, calculate, and analyze their academic performance with an intuitive interface.

## Features

- **Dashboard**: View current CGPA, latest semester GPA, total credits, and performance status
- **GPA Calculator**: Calculate semester GPA by adding courses with credits and grades
- **CGPA Tracker**: View and manage all semesters with cumulative GPA calculation
- **Analytics**: Visualize GPA progression with charts and performance insights
- **Goal Predictor**: Calculate required GPA to achieve target CGPA
- **Settings**: Customize grading scale (4.0/5.0) and theme (light/dark mode)
- **Local Storage**: All data persists in browser storage for offline access

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

1. Navigate to the GPA Calculator to add your first semester
2. Enter course names, credits, and grades
3. Save the semester to track it in your CGPA
4. View analytics to see your performance trends
5. Use the goal predictor to plan future semesters

## Technology Stack

- React 18
- TypeScript
- Vite
- CSS3 with responsive design
- LocalStorage API

## Responsive Design

The application is fully responsive and works seamlessly on:
